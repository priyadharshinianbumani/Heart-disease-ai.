# Early Heart Disease Risk Detection

This project is an AI-based system to detect early heart disease risk using wearable passive signals.

## Contents

- `backend/` - Python ML code
- `frontend/` - Web app code
- `images/` - Screenshots & diagrams
- `ppt/` - Presentation slides
